Description

Note: GHC.Generics exports a number of names that collide with Control.Lens.

You can use hiding or imports to mitigate this to an extent, and the following imports, represent a fair compromise for user code: 

 import Control.Lens hiding (Rep)
 import GHC.Generics hiding (from, to)

You can use generic to replace from and to from GHC.Generics, and probably won't be explicitly referencing Rep from Control.Lens in code that uses generics.

If you're using a version of GHC older than 7.2, this module is compatible with the http://hackage.haskell.org/package/generic-deriving generic-deriving package.

Generic Traversal

tinplate :: (Generic a, GTraversal (Rep a), Typeable b) => Simple Traversal a b

A Generic Traversal that visits every occurrence of something Typeable anywhere in a container. 

>>> allOf tinplate (=="Hello") (1::Int,2::Double,(),"Hello",["Hello"])
True

>>> mapMOf_ tinplate putStrLn ("hello",[(2 :: Int, "world!")])
hello
world!
